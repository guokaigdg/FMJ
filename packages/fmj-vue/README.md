
# 学习VUE练习使用

## 🚀 技术栈
```
脚手架: vue/cli 
前端框架: vue 2.0 ( 没有上3.0是因为Element U 兼容问题)
UI框架: Element UI 
路由:vue-router
状态存储: Vuex
```
## 💻 项目依赖安装

```
yarn install
```

### ⚙️ 项目开发环境

```
yarn dev
```

### 📦 项打包目编译

```
yarn build
```

###  🛵 Lints和修复文件

```
yarn lint
```
